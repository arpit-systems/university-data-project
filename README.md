# 📊 University Data Management & Analytics System

## 🔹 Overview

This project demonstrates a complete relational database system built using SQL. It includes database design, relationships, and analytical queries to extract insights.

## 🔹 Database Design

* Entity Tables: student, course, professor
* Junction Tables: enrollment, teaching
* Relationships: Many-to-Many
* Keys: Primary Keys, Foreign Keys, Composite Keys

## 🔹 SQL Concepts Used

* SELECT, WHERE, JOIN
* GROUP BY, COUNT, HAVING
* ORDER BY, DISTINCT
* Subqueries, CASE

## 🔹 Key Analysis

* Student-Course mapping
* Course popularity analysis
* Students with multiple enrollments
* Courses with no enrollments
* Full relational mapping (Student + Course + Professor)

## 🔹 Tech Stack

* SQL (MySQL)
* Relational Database Design

## 🔹 Project Purpose
To showcase SQL proficiency through relational database design, data modeling, and solving real-world analytical problems using structured queries.
